# TLV 広告収益モデル

---

## 1. 計算式

```
広告収入 = 総インプレッション × (CPM / 1000) × (1 + CTR)
```

| パラメータ | デフォルト | 備考 |
|-----------|-----------|------|
| CPM | ¥450 / 1,000 imp | 日本動画広告の保守的中値 |
| CTR | 0.8% | 収益補正係数として使用 |
| 手入力上書き | 0（無効） | Inputs で直接円額指定可 |

Stripe 手数料は **広告収入の 3.6%** をコスト側で控除。

---

## 2. インプレッション算出

### ショート動画
```
imp = 月間再生数 × 広告imp/再生
    = 80,000 × 1.1 = 88,000 imp
```

| 入力 | 値 |
|------|-----|
| 月間再生数 | 80,000 |
| 広告 imp/再生 | 1.1 |
| 平均視聴時間 | 32 秒 |
| 平均再生率 | 72% |

**広告収入（ショート単体）:** 88,000 × 0.45 = **¥39,600**

### 通常動画
```
imp = 35,000 × 2.4 = 84,000 imp
```

| 入力 | 値 |
|------|-----|
| 月間再生数 | 35,000 |
| 広告 imp/再生 | 2.4（プレロール+ミッドロール想定） |
| 平均視聴時間 | 240 秒 |
| 平均再生率 | 38% |

**広告収入（通常単体）:** 84,000 × 0.45 = **¥37,800**

### ライブ配信
```
imp = 月間視聴時間(h) × 広告imp/h
    = 4,800 × 3.5 = 16,800 imp
```

| 入力 | 値 |
|------|-----|
| 月間視聴時間 | 4,800 h |
| 広告 imp/h | 3.5 |
| 平均同時視聴 | 80 人 |

**広告収入（ライブ単体）:** 16,800 × 0.45 = **¥7,560**

## 3. 合計（ベースケース）

| 種別 | imp | 収入（円） | 構成比 |
|------|-----|-----------|--------|
| ショート | 88,000 | 39,600 | 46.2% |
| 通常 | 84,000 | 37,800 | 44.1% |
| ライブ | 16,800 | 7,560 | 8.8% |
| **合計** | **188,800** | **¥85,640** | 100% |

（CTR 補正込みの丸め後合計）

## 7. 月次還元判断への入力

| エンジン入力 | 算出 |
|-------------|------|
| `total_ad_revenue` | 種別合計または手入力 |
| `short_revenue` | ショート imp × CPM |
| `normal_video_revenue` | 通常 imp × CPM |
| `live_revenue` | ライブ imp × CPM |

`creator_monthly_revenue` は各クリエイターの広告貢献額（種別按分または直接計測）を入力し、段階制還元の基礎とします。
