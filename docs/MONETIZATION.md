## 1. 基本方針

| 原則 | 内容 |
| --- | --- |
| **利益が先** | 還元は利益成立後 |
| **再投資** | 還元 · 新人支援 · イベント · インフラ |
| **条件達成型還元** | 固定還元率ではなく、達成条件を満たしたクリエイターに業界最高水準を目指す |
| **インフラ意識** | ライブ帯域 · エンコード · 保存 · CDN コストをモデルに組み込む |

## 3. 条件達成型還元（核心）

### 3.1 考え方

```
還元率を先に決めない
    ↓
クリエイター活動 → 収益 · コスト · 品質を計測
    ↓
利益が成立する条件を満たしたクリエイターのみ
    ↓
高還元ティア（業界最高水準を目指す）
```

### 3.2 還元条件の例（設計案 · 数値は未確定）

| 条件カテゴリ | 例 |
| --- | --- |
| **収益貢献** | 月次 net プラットフォーム利益が閾値以上 |
| **品質** | ポリシー違反なし · 著作権クレーム率が低い |
| **継続** | 一定期間の active 配信 / 投稿 |
| **視聴者満足** | 通報率 · 離脱率 · 低評価率が閾値以下 |
| **コスト効率** | ライブ時間あたり infra コストが許容内 |

**未達時:** 標準還元（または還元対象外）— **赤字をプラットフォームが肩代わりしない**

### 3.3 還元ティア（概念）

| ティア | 概要 |
| --- | --- |
| **Standard** | 全参加クリエイターのベース |
| **Proven** | 利益貢献条件を満たした層 |
| **Top Contributor** | 高利益 · 高品質 · 継続 — **最高還元候補** |

数値 · パーセンテージは **運用データ取得後** に決定（本書では固定しない）。

## 4. 利益の再投資配分（方針）

| 用途 | 優先 |
| --- | --- |
| 条件達成型還元アップ | 高 |
| 新人支援 · Creator Incubator | 中 |
| イベント · キャンペーン | 中 |
| インフラ · 不正対策 · CDN | 高（必須） |
| 運営人件 · Ops ツール | 中 |

**ルール:** 再投資前に **プラットフォーム利益の黒字** を確認。

---

## 5. コストモデル（ライブ）

| コスト項目 | 制度への反映 |
| --- | --- |
| 配信帯域 · エンコード | 30 分無料 + 条件延長 |
| 同時接続数 | オーバーゲージ上限 |
| 録画 / DVR 保存 | Proven ティア以上で段階提供（将来） |
| 不正 · ボット視聴 | Creator Score 減点 · 還元停止 |

---

## 6. 禁止・非採用

| 項目 | 理由 |
| --- | --- |
| 全員 90%+ 還元の常時宣言 | インフラ赤字 · 持続不可 |
| 還元率のみで獲得するマーケ | 低品質 · 不正誘因 |
| ライブ無制限延長（無条件） | コスト爆発 |
| クリエイター間の収益率公開ランキング | 摩擦 · 不正競争 |
